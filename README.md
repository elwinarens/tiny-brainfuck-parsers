Tiny brainf*ck parsers
======================

Tiny, fully functional Brainf*ck parser implementations optimized for code size.

### Tiny BF JavaScript parser

Usage when including B.min.js:
```JS

output = B('++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.');

console.log('output:', output); // output: Hello World!

```
